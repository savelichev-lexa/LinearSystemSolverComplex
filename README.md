# LinearSystemSolverComplex
Решение систем линейных алгебраических уравнений (СЛАУ) с комплексными числами
Данный проект реализует три численных метода решения СЛАУ с комплексными коэффициентами:

Методы решения
Метод простой итерации (Якоби) — итерационный метод, использующий одновременное обновление всех переменных на каждой итерации.

Метод Гаусса-Зейделя — итерационный метод, при котором обновленные значения переменных используются сразу же в текущей итерации, что обеспечивает более быструю сходимость.

Метод Гаусса — прямой метод, основанный на последовательном исключении неизвестных (прямой и обратный ход).

Особенности
Работа с комплексными числами (std::complex<double>)

Автоматическая генерация тестовых матриц размером 50×50, 125×125 и 250×250

Сравнение производительности методов с замером времени выполнения

Сохранение результатов решения в отдельные файлы для каждого метода и размера матрицы

Формат входных данных
Матрицы генерируются со специальной структурой, обеспечивающей диагональное преобладание для сходимости итерационных методов.

Формат выходных данных
Результаты сохраняются в файлы:

result_Jacobi{N}.txt

result_GaussSeidel{N}.txt

result_Gauss{N}.txt
